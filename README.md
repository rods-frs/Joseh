# Joseh - Open Source AI Voice Assistant (CURRENT ON ALPHA)

A lightweight, open-source Python voice assistant inspired by Alexa. Joseh listens to voice commands, processes natural language using AI, and executes various actions including music control, system updates, and more.

## Features

🎤 **Voice Command Recognition** - Speak naturally and Joseh understands your commands via Google Speech Recognition

🤖 **NLP-Powered Intent Recognition** - Uses spaCy for natural language processing to understand complex, multi-step commands

🎵 **Spotify Integration** - Full control over playback (play, pause, skip, previous, current track info)

🖥️ **System Commands** - Execute system updates on Linux (Fedora, Ubuntu)

📅 **Utility Functions** - Get current date and time

🔧 **Command Parsing** - Understands complex commands like "pause and then skip" or "play next, and then tell me the date"

🎙️ **Text-to-Speech** - Responds with spoken feedback using pyttsx3

## Requirements

- **Python** 3.12
- **Operating System**: Linux (Arch-based or Ubuntu/Fedora systems)
- **Dependencies**: See [Installation](#installation)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/rods-frs/Joseh.git
cd Joseh
```
2. Execute the main code:
```bash
python3 ./mainV4.py
```
